# wireguard-instalacion-configuracion
Este proyecto documenta el proceso completo de instalación, configuración y puesta en marcha de WireGuard en un servidor Linux, junto con la conexión de clientes en Windows y Android.  Incluye instrucciones paso a paso, ejemplos de archivos de configuración y recomendaciones de seguridad para desplegar una VPN rápida y segura basada en WireGuard.
